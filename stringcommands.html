String Commands

String: Wildcard Pull

You can use this command with a button trigger to pull a wildcard value. More about wildcards

Name Type Description
Variable String Variable name to store the pulled wildcard value under
Wildcard number Int This refers to which wildcard you wish to get.
0 = first *, 1 = second *, etc.
Turn to Real boolean Whether you expect a real value (=number) or a string.

String: Get Trigger Type

Returns what kind of trigger activated the button. This button can be used to know which Math: Trigger Pull value to use.

Possible returns (including capitalization):

  • Bits
  • Sub
  • Redeem
  • Chat message
  • Host
  • Raid
  • Redeem with message
  • Extension
  • Gifted sub
  • Other - returned for any trigger that is not listed (through your Stream Deck, Command Trigger or Shortcut Trigger)
Name Type Description
Variable String Variable name to store the returned trigger under.


String: Length

Returns the length of a string.

  • "Hello" will return 5
  • "" (empty string) will return 0.
Name Type Description
Variable String Variable name to save the length value into
String Variable String Variable name containing the string you wish to know the length of

String: Format

Divides a string into multiple lines. The new line will be compensated with \n.
Very useful command to use together with Math: Trigger Pull for Bits or Redeem points with a message. You can force the viewer's message to be split into multiple lines in your Text GDI+ source.

Name Type Description
Variable String Variable name to save the formatted string
String String String to format. Use /$variable$/ if your string is inside another variable.
Max Width String An arbitrary number. This is technically a pixel width, but you will have to test to see what you receive back.
Max Word Length String Amount of characters to allow before cutting the word in the middle.

String: Digits

Removes all letters and special symbols from a string.

  • "Hello" will return "" (empty string)
  • "1a2b3c" will return "123"
  • "$12.50" will return "12.50".
Name Type Description
Variable String Variable name containing the string you wish to convert to digits only

String: Letters

Removes every non letter from a string. This includes spaces and line breaks.

  • "12.50 USD" will return "USD".
  • "Hello world!" will return "Helloworld".
Name Type Description
Variable String Variable name containing the string you wish to convert to a letter only string

String: Replace

Searches a string for a specified value and returns a new string where all the occurrences of the specified value are replaced. Case sensitive.

  • {"Hello","l","/"} will return "He//o".
  • {"Hello World"," ","-"} will return "Hello-World".
  • {"Mr.Black has a black car and a black cat.", "black", "white"} will return "Mr.Black has a white car and a white cat."
Name Type Description
Variable String Variable name containing the original string.
Replace String Value that will be replaced by the new value.
New String String The value to replace the search value with. Can be empty to completely remove it.

String: Lower Case

Converts a string to lowercase letters.

  • "HELLO World" will return "hello world"
Name Type Description
Variable String Variable name containing a string to convert to lowercase letters

String: Upper Case

Converts a string to uppercase letters.

  • "hello world" will return "HELLO WORLD"
Name Type Description
Variable String Variable name containing a string to convert to uppercase letters

String: Position

Returns the position of the first occurrence of a specified value in a string. Case sensitive.

  • {"hello and hello world","hello"} will return 1
  • {"Hello and hello world","hello"} will return 11

This is a very useful command to use together with String: Clamp command, as you can easily remove a part of a text.

Name Type Description
Variable String Variable name to save the position
String String String to search
Find String String String to search for

String: Count

Returns the number of occurrences of a character/string within a string. Case sensitive.

  • Searching for "Ab" within a string "AbAbAbAb" will return 4.
  • Searching for "33" within a string "333 and 33 and 33333" will return 7.
  • Searching for "black" within a string "Mr.Black has a black car and a black cat." will return 2.
Name Type Description
Variable String Variable name to save the count
String String String to search
Count String String String to search for and count

String: Clamp

Extracts a part of a string and returns the extracted part. You need to specify the index of the first character and the amount of characters to extract.
This is a very useful command to use together with String: Position command. This way you can easily remove a part of a text.
Do not use String:Clamp on real values and Math:Clamp on strings, as this will result in an error and LB crashing.

Name Type Description
Variable String Variable name containing the string to clamp
Start Real Where to start the clamp
Count Real Amount of characters to include in the clamp

String: Real to String

Turns a real value into a string. Useful if you want to execute String commands.

  • -1.2 will return "-1.20"
  • 1.12345 will return "1.12"
Name Type Description
Variable String Variable name to save the real value
Source String Variable name containing the real value to stringify

String: Sound Name Pull

Pulls the names of the sounds that were played by this button.
The name returned will be a file name excluding the path (E:\Stream\sound.ogg will return sound.ogg)

Name Type Description
Variable String Variable name to store the pull in
Array Number Real Which sound to get in order they appear in the list.
0 will be the one at the top of the list.
