Other Commands

Set Clipboard

Changes the content of your clipboard to whatever is in the string box
You can then use Right Click-Paste or CTRL+V to paste it anywhere else.

Name Type Description
String String The string you wish to save in your clipboard.

Load Clipboard

Loads the contents of your clipboard (after using CTRL+C) into a variable.
You can use this command together with Execute Program command to copy any result from your .bat file into LioranBoard. Just insert |clip at the end of your code in your .bat file.

Name Type Description
Variable String Variable name to save the string under.

Trigger Button

Triggers any button of a given ID. You can right click on a button to retrieve its button ID.
If the button ID does not exist, sends an extension trigger instead.

Buttons can not have the same ID. If you copy and paste a button, it will have a new ID unless you deleted the original one.

Name Type Description
Button ID Int ID of a button you want to activate.
Queue-Able boolean If set to true and the button is already activated, it will wait for it to finish and trigger the button again.
Will get queued as many times as you execute the command.
If set to false and the button is already activated, nothing will happen.
Hidden Delay Int This exists to make a button happen without greying it out in case the button is showing valuable info.
The delay is hidden and cannot be cancelled (unless you use Reset or press Done in your Receiver).

Modify button

Changes the appearance of a button (color, text, image and border).
All changes are temporary, the button will return to its original state if you reset or close and reopen Receiver.
If you wish to reset the button back to its default appearance, set the values to {buttond_id, -1, -1, -1, notext, leave empty, -1}.

Name Type Description
Button ID Int Button ID to change the appearance of
Red * Int Red value, 0-255.
Green * Int Green value, 0-255.
Blue * Int Blue value, 0-255.
Text String New button text. Leave blank or enter notext to use default value.
Using # symbol for a new line might be a bit buggy.
Picture String File name of a picture to use. Must be formatted as image.png
and placed in Lioranboard Receiver/image folder.
Do not include path. Leave empty to use default value.
Border Size Int Size of the button border, between 0-8. Only applies if a picture is included.
Use -1 for default border size and 0 for no border.

Clear Button

Clears the given ID or all the buttons (in all decks). You can choose to clear queued buttons as well.
If you clear all buttons, you can specify whether you want to clear the button performing this command as well.

Name Type Description
Button ID Int Button ID to clear
Ongoing boolean Specify whether the ongoing buttons should be cleared.
Queue boolean Specify whether the queued buttons should be cleared.
Self boolean If you selected all in your Button ID field, you can choose whether you want to
clear this button as well.
Does not work if the button ID is itself (in that case use False in Ongoing field).

Open URL

Opens any URL in your default browser. Must contain http or https to properly work.

Name Type Description
URL String URL to open, must be a full link. For example https://twitch.tv.

Execute Program

Launches any program from your PC. It can also run some files.
If LioranBoard is running as admin, any program you launch through this command will also run as admin.

Name Type Description
File Path String Path to the .exe file you wish to run. Must be a full path.
For example: E:\lioranboard\LioranBoard Stream deck(PC)\LioranBoard Stream Deck.exe

Command Line

Can execute a command line just like you do in Command Prompt. Only one line of code.
If you want to execute more than one line of code, I suggest converting it to .bat file and using Execute program command instead.
If LioranBoard is running as admin, commands you perform will also run as admin.
You can use |clip at the end of your command to copy the result to your clipboard and then use Load Clipboard command to load it into LioranBoard.

This command can be a hit or miss. Some commands don't get executed or the CMD prematurely closes. If you encounter this problem, try converting it into batch file instead.

Name Type Description
Command String Command you wish to execute.

Send to Extension

You can use this command to send data to an extension.
If you wish to create your own extension, check out the readme.txt file in your lioranboard/extension folder..
The values change depending on which extension you use.
If you just installed a new extension, but do not see its name in the list, you need to restart Transmitter or close/reopen LioranBoard.

Name Type Description
Extension String Name of the extension.
Various values String These change depending on the specific extension.
If you selected manual in the previous field, the JSON String must be in JSON format with {} on both sides.
For example: {"datatype":0, "value":321.2, "name":"Robert"}

Copy Local File

This command lets you copy a file from the Receiver folder and paste it in any of its folders.
This serves to make a life easier for extension creators, allowing them to transfer extension files to the image and sound folder.
Will not replace the file if one with the same name is already present.
As of now it's not possible to copy files from outside of the Receiver folder.

Name Type Description
Local File Path Local Path Full path including the file name you wish to copy.
Local Folder Destination Local Path Where you want to copy the file.
You can use sound\ for sound folder and image\ for image folder.

Popup Message

This command lets you show a window popup message.
Useful for extension creators to be able to easily notify the user of important events.
Will not freeze the Receiver, but it's still recommended to use it rather sparingly.

Name Type Description
Message String The message you want to display.
You can use # to add a new line and \# to display #.

Alert Message

This will show a small yellow alert message at the bottom of the Receiver.

Name Type Description
Message String The message you want to display.

Overtime

This command lets you add a cooldown to a button, artificially increasing the duration of the button and preventing it from triggering again.
This is a useful command if you want to make sure your notification alerts do not overlap or if you want to prevent viewers from triggering your buttons over and over again without any cooldown in between.

Name Type Description
Overtime Duration (ms) Int The amount of time the overtime (cooldown) should last.
The countdown starts as soon as the button is triggered.

Comment

This is helpful if you want to leave a comment explaining what each command/section does in your button.

Name Type Description
Comment String Whatever useful information you want to put in here for you or other users to read.
