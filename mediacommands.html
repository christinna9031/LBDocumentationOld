OBS Media Commands

Change Audio Monitor Type

Changes the monitoring type of an audio source. Same option as in OBS-Audio Mixer-Cogwheel-Advanced Audio Properties.

Name Type Description
sourceName String Audio source name
monitorType dropdown menu Type of monitoring to use

Play/Pause Media

Requires OBS Websocket 4.9

Play or pause a media/VLC source.

Name Type Description
sourceName String Name of the source to play or pause.
playPause Boolean True to pause and false to play

Restart Media

Requires OBS Websocket 4.9

Restart a media/VLC source.

Name Type Description
sourceName String Name of the source to restart.

Stop Media

Requires OBS Websocket 4.9

Stop a media/VLC source.

Name Type Description
sourceName String Name of the source to stop.

Next Media

Requires OBS Websocket 4.9

Play next track. Can only be a VLC source.

Name Type Description
sourceName String Name of the track to skip.

Previous Media

Requires OBS Websocket 4.9

Play previous track. Can only be a VLC source.

Name Type Description
sourceName String Name of the track to go back.

Set Media Time

Requires OBS Websocket 4.9

Seek to position in a currently playing source. Can be a media or VLC source.

Name Type Description
sourceName String Name of the source
sourceName Int Time in milliseconds
