File Commands

Introduction

LioranBoard can natively read and modify only .ini files in its own folder. There are two possible default locations for the .ini files:

  1. \lioranboard\LioranBoard Receiver - this is the primary location
  2. C:\Users\USERNAME\\Local\LioranBoard - this is the secondary location. Files are saved here instead if something blocks LioranBoard from saving them in their primary location.

Each file needs to have a specific structure for LioranBoard to read it, containing [section] and key="value".

Saving to .txt files

There is a way to save a value to a .txt file:

  1. Create a text.bat file with this content:
    @echo off
    echo %~2 >> %1.txt
  2. Save the file in your lioranboard/LioranBoard Receiver folder.
  3. Use a Command Line command in LB: text.bat FILENAME "VALUE"

  4. The .txt file will be created in your Receiver folder if it doesn't exist yet. If it does exist, it will just add another line to it.

Note: If your text input already contains double quotes, it will split it and only save the first half. Single quotes are OK.

File: Save Value

Saves a real value to .ini file.
Can be used to permanently save a value (or a stack), so you can retrieve it even after you close or restart LioranBoard.
Make sure you're using the correct command, as it will not properly save if you use File: Save Value on a string.

Name Type Description
File Name String Name of the file. Only .ini files are accepted. Will create one if it doesn't exist.
Do not include path. You do not need to include the .ini extension.
Section String The section to save the key under.
Key String The name of the value to be saved.
Variable String Variable to be saved as the key value.

File: Load Value

Loads a real value from .ini file. Will return 0 if it doesn't exist.
Can be used to retrieve previously saved values after you close/restart LioranBoard. Can be also used to keep a list of file paths and use it together with Math: Random command to display a random picture or play a random sound/video.
Make sure you're using the correct command, as it will not properly load if you use File: Load Value on a string.

Name Type Description
File Name String Name of the file. Only .ini files are accepted.
You do not need to include the .ini extension.
Section String The section to load the key.
Key String The name of the key to be loaded.
Variable String Variable name you want to save the loaded value under.

File: Save String

Saves a string to .ini file.
For example, it can be used to save a list of followers or subscribers from the given day.
Make sure you're using the correct command, as it will not properly save if you use File: Save String on a real value.

You do not need to include the .ini extension.

Key String The name of the value to be saved.
