Statement Commands

All previous Compare Commands have been deprecated and replaced with Statement Commands. You can tick the Auto Replace 'Compare' to 'Statement' checkbox at the top of your deck to automatically replace your deprecated commands with the new statement commands whenever you go edit your buttons (you will be able to still review the changes).

Conditional statements are used to perform different actions based on different conditions. If a condition is true, you can perform one action and if the condition is false, you can perform another action.

If - Commands to be executed if the specified condition is true.
Else - Commands to be executed if the specified condition is false.
And - All specified conditions need to be true.
Or - At least one of the specified conditions needs to be true.

Statement: If

If statement is used to specify a block of commands to be executed if a condition is true.
Else statement is executed if a condition is false.
You can think of it as 'If something is true, then do something, otherwise do something else.'.
You can combine If statements with And and Or statements and also create other nested If statements.

Name Type Description
Var./real/"string" Variable, real or string Left side of the comparison you want to evaluate.
Compare Type Dropdown menu Operator you wish to use to compare left and right side.
Var./real/"string" Variable, real or string Whatever you want to compare the left side with.

Example 1: If a viewer's name is 'Lioran', say 'Hello Lioran!'. Else If viewer's name is 'Melonax', say 'Hello Melonax!'. Else (if viewer's name is not Lioran or Melonax), say 'Hello random person!'.

Example 2: If a viewer's badge is subscriber OR vip OR moderator (at least one condition needs to be met), then allow them to change your scene. Else say 'You're not privileged enough to change the scene!

Example 3: If my microphone is not muted AND current scene is 'BRB', send a pop up alert 'Your mic is not muted!'

Example 4: If I'm currently streaming, then [If my current scene is 'main camera', then take a screenshot. Else (if my current scene is NOT 'main camera') switch to main camera and take a screenshot.] Else (if I'm not streaming) say 'Sorry, the stream is offline!'.'

Statement: And

Statement: And is used in junction with a Statement: If. It allows you to compare multiple conditions where ALL of them need to be true. If one of them is false, the whole If statement will be also false.
Statement: And must be placed inside Statement: If as the first command.

Name Type Description
Var./real/"string" Variable, real or string Left side of the comparison you want to evaluate.
Compare Type Dropdown menu Operator you wish to use to compare left and right side.
Var./real/"string" Variable, real or string Whatever you want to compare the left side with.

Example: If my microphone is not muted AND current scene is 'BRB', send a pop up alert 'Your mic is not muted!'

Statement: Or

Statement: Or is used in junction with a Statement: If. It allows you to compare multiple conditions where at least one of them needs to be true. Statement: Or must be placed inside Statement: If as the first command.

Name Type Description
Var./real/"string" Variable, real or string Left side of the comparison you want to evaluate.
Compare Type Dropdown menu Operator you wish to use to compare left and right side.
Var./real/"string" Variable, real or string Whatever you want to compare the left side with.

Example: If a viewer's badge is subscriber OR vip OR moderator (at least one condition needs to be met), then allow them to change your scene. Else say 'You're not privileged enough to change the scene!

Statement: Re-enable if

Commands inside the statement will be repeated for x number of times with x milliseconds of delay between each repeat IF the condition is true. Else they will be executed only once.
Statement: Re-enable if can contain AND and Or statements as well.
You can create some really cool effects with this command.

Name Type Description
Var./real/"string" Variable, real or string Left side of the comparison you want to evaluate.
Compare Type Dropdown menu Operator you wish to use to compare left and right side.
Var./real/"string" Variable, real or string Whatever you want to compare the left side with.
Reactivate (ms) Int Delay between each re-enable. If set to 0, it will repeat indefinitely.
Amount Int How many times you want to re-enable the commands. Set to -1 for unlimited amount.

Example 4: Continuous yaw rotation of an image source in OBS.


Repeat

This command instantly repeats all the commands inside the block for a specified number of times. This is useful for instantly retrieving multiple values from File: Load Value/String or from Stack: Pull command.

Name Type Description
Repeat amount Int (number) How many times you wish to repeat commands inside the block.

Example: Instantly pull all the values from a stack and save them in separate variables.

Statement: Exit if

If the condition is true, it terminates all the commands and prevents them from happening until the button is pressed again.
Will not cancel button macros or sounds attached to the button.
You can use this command in junction with Statement: And and Statement: Or commands.

Name Type Description
Var./real/"string" Variable, real or string Left side of the comparison you want to evaluate.
Compare Type Dropdown menu Operator you wish to use to compare left and right side.
Var./real/"string" Variable, real or string Whatever you want to compare the left side with.

Statement: Break if

If the condition is true, this command will break out of a repeat or re-enable statement, cancelling every command inside of the block if it hasn't been executed yet. Re-enable statement will be disabled as well. If you place the command inside multiple repeat or re-enable statements, it will only break out of its parent statement.
You can use this command in junction with Statement: And and Statement: Or commands.

Name Type Description
Var./real/"string" Variable, real or string Left side of the comparison you want to evaluate.
Compare Type Dropdown menu Operator you wish to use to compare left and right side.
Var./real/"string" Variable, real or string Whatever you want to compare the left side with.

Example: Attempt to find a winning number 72 in the provided stack by randomly pulling its values. Once a match is found, inform the chat how many tries it took.
